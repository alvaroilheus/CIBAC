# 📰 CIBAC

**Projeto da matéria de HTML, CSS e JavaScript - SENAI**

---

## 📖 Sobre o Projeto

O **CIBAC** é uma proposta de portal de notícias voltado para a rede SENAI. Desenvolvido de forma livre, o objetivo deste projeto é aplicar na prática os conceitos de desenvolvimento web frontend aprendidos em sala de aula, criando um ambiente informativo, responsivo e de fácil navegação para os alunos e colaboradores da instituição.

##  Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias web fundamentais:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout e responsividade.
* **JavaScript:** Interatividade e dinamicidade da página.

##  Funcionalidades

*  Aba de navegação rápida.
*  Animações 3D.
*  Totalmente responsivo para diferentes tamanhos de tela.
*  Efeitos visuais para melhorar a experiência do usuário.
*  Exibição de notícias em destaque.
*  Categorização de matérias (Tecnologia, Indústria, Cursos, etc.).
*  Navegação para contato e parceiros(Embraer, General Motors, etc.).
*  Galeria de videos e fotos.

##  Design Visual

A identidade visual do site foi pensada para ser limpa e focar na legibilidade das notícias.
